## learning
建立神经网络的主要步骤：  
  1. 定义模型结构（例如输入特征的数量）  
  2. 初始化模型的参数   
  3. 循环：  
   3.1. 计算当前损失（正向传播）  
   3.2. 计算当前梯度（反向传播）  
   3.3. 更新参数（梯度下降）  
  4. 训练与测试  
  5. 建立model整合函数
